\f0\fs24 \cf0 # Credit Card Recommendation Assistant \uc0\u55357 \u56499 \
\
An AI-powered credit card recommendation app that helps users find the best credit cards based on their specific needs and spending habits.\
\
## Features\
\
- \uc0\u55357 \u56492  Natural language chat interface\
- \uc0\u55358 \u56598  Powered by OpenAI GPT-4\
- \uc0\u55357 \u56522  Database of 350+ credit cards\
- \uc0\u55356 \u57263  Top 3-5 personalized recommendations\
- \uc0\u55357 \u56589  Smart matching based on user queries\
\
## How It Works\
\
1. User asks a question (e.g., "What's the best card for travel rewards?")\
2. GPT-4 analyzes all credit cards in the database\
3. Returns top recommendations with explanations\
4. Shows why each card is a good match and highlights key perks\
\
## Deployment on Streamlit Cloud\
\
1. Push this repository to GitHub\
2. Go to share.streamlit.io\
3. Sign in with GitHub\
4. Click "New app"\
5. Select your repository\
6. Add your OpenAI API key in Secrets\
7. Deploy!\
\
## Data Format\
\
Your credit_cards.csv should have two columns:\
- card_name\
- perks_summary\
